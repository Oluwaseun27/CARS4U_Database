 CARS4U_Database

Project Overview

This project involved designing and developing a Vehicle Rental Management Application using Oracle Application Express (APEX). The application was built to support the end-to-end management of vehicle rentals, including customer records, vehicle inventory, bookings, and rental status tracking.

The project demonstrates the use of low-code enterprise application development, database design, and process automation to support real-world business operations.


OBJECTIVES
Design a structured database to manage vehicle rental operations

Build a functional web-based application using Oracle APEX

Enable efficient tracking of vehicles

Apply data validation and role-based access where appropriate

TECHNOLOGIES USED

Oracle Application Express (APEX)

Oracle Database (SQL)

PL/SQL (for validations and business logic)

Entity-Relationship (ER) Modelling
